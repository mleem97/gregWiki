## Sponsorship Tiers

### ☕ $1 / month – The Coffee Supporter
- GitHub sponsor badge.
- Discord `Supporter` role.
- Eternal gratitude.

### 🥉 $5 / month – Bronze Backer (Active Player)
*Includes all previous perks, plus:*
- Credits entry in `gregModmanager` ("Special Thanks / Sponsors").
- Access to sponsor-only Discord channels for WIP and development updates.

### 🥈 $15 / month – Silver Tester (Early Access)
*Includes all previous perks, plus:*
- Early access to pre-release and beta builds.
- Roadmap voting access for Silver+ backers.

### 🥇 $25 / month – Gold Developer (Pro Modder)
*Includes all previous perks, plus:*
- Priority handling for framework bug reports.
- Official Gold sponsor mention in project READMEs.

### 💎 $50 / month – The Ecosystem Architect (Premium/VIP)
*Includes all previous perks, plus:*
- Featured mod spot on `datacentermods.com`.
- Private 1-on-1 Discord support.

## Automation
Sponsorship status is synchronized via GitHub Actions using the `.github/workflows/sponsor-tier-sync.yml` workflow, which updates roles on Discord and the official website.
