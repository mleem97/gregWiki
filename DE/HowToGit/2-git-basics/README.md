## Zusammenfassung

Du solltest jetzt in der Lage sein:
- Ein Repository zu erstellen oder zu klonen
- Änderungen vorzunehmen und zur Staging-Area hinzuzufügen
- Commits anzulegen und die Historie zu durchsuchen
