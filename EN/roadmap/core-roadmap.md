# Core Roadmap

## Direction

This project maintains the `gregCore` mod framework for Data Center and compatible IL2CPP Unity games. Development focuses on:

1. Framework stability
2. Mod compatibility with game updates
3. Developer experience improvements

## Short-term Goals

- [ ] Fix reported bugs
- [ ] Maintain compatibility with game updates
- [ ] Improve build system

## Medium-term Goals

- [ ] Improve documentation
- [ ] Expand test coverage
- [ ] Developer tooling

## Long-term Goals

- [ ] IL2CPP helper tool for easier reference extraction
- [ ] Better AI integration support
- [ ] Cleaner mod architecture

## Disclaimers

- No hard release schedule
- Goals subject to change based on game updates
- Contributions welcome

## Future Tooling

A **dedicated helper tool** is planned to simplify the IL2CPP reference extraction process for AI-assisted modding workflows. This would reduce manual setup burden when working with AI coding agents.
