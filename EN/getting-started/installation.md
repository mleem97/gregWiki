# 📥 Installation & Prerequisites

Setting up gregCore correctly is the most important step to ensure your mods run stably and don't crash the game. Follow this guide carefully.

## 🛠️ Hardware & Software Requirements

- **Game**: *Data Center* (Steam version recommended).
- **OS**: Windows 10/11 (x64).
- **Runtime**: [.NET 6.0 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/6.0) (usually included with MelonLoader).
- **Mod Loader**: [MelonLoader v0.6.1+](https://github.com/LavaGang/MelonLoader/releases).

## 1️⃣ Base Installation (End-User)

If you just want to run mods that require gregCore, follow these steps:

1.  **Install MelonLoader**: 
    *   Download the `MelonLoader.Installer.exe`.
    *   Select your `Data Center.exe`.
    *   Ensure "Latest" or "v0.6.1" is selected.
    *   Click **Install**.
2.  **Download gregCore**:
    *   Get the latest `gregCore.dll` and the `gregDependencies.zip` from the [Releases](https://github.com/mleem97/gregCore/releases).
3.  **Deployment**:
    *   Copy `gregCore.dll` into the `Mods` folder in your game directory.
    *   Extract `gregDependencies.zip` so that you have a folder named `gregDependencies` inside the `Mods` folder.
    *   *Resulting Structure:*
        ```text
        Data Center/
        ├── MelonLoader/
        ├── Mods/
        │   ├── gregCore.dll
        │   └── gregDependencies/
        │       ├── LiteDB.dll
        │       ├── MoonSharp.Interpreter.dll
        │       └── ...
        ```
4.  **Verification**:
    *   Launch the game.
    *   Check the console window. You should see the ASCII banner: `gregCore Modding Framework`.

## 2️⃣ Development Setup (Modder)

To write mods using gregCore, you need additional tools:

### For C# Modding
- **IDE**: Visual Studio 2022 (Community is fine) or JetBrains Rider.
- **Reference Assemblies**: You need the unhollowed game assemblies generated by MelonLoader. Run the game once with gregCore installed; it will generate them in `MelonLoader/Il2CppAssemblies`.

### For Scripting (Lua/JS/Python)
- **Editor**: [VS Code](https://code.visualstudio.com/) is highly recommended.
- **Extensions**:
    - `sumneko.lua` for Lua intellisense.
    - `ms-python.python` for Python.
    - `esbenp.prettier-vscode` for JavaScript.

## ⚠️ Troubleshooting Installation

### "MissingFieldException: SystemGameLoaded"
This usually means you have an old version of `gregCore.dll` but a mod was compiled against a newer version. Always ensure `gregCore` and your mods are up to date.

### "gregDependencies not found"
gregCore will log a warning if it cannot find the dependency subfolder. While some features (like basic hooks) might work, language bridges (Lua/JS) and data persistence will fail.

### Antivirus False Positives
Since gregCore uses Harmony to patch memory, some aggressive antivirus software might flag it as a "Trojan" or "Injector". You may need to add an exclusion for your game folder.
