# Official Mods & Examples

The following mods are developed and maintained by **teamGreg**. They serve both as useful features for players and as reference implementations for developers to learn from.

---

## 🛠️ Essential Tools
- **[HexViewer](./hex-viewer)**: A powerful in-game inspector for viewing raw hexadecimal hardware data.
- **[IPAM (Network Management)](./ipam/index)**: Advanced IP address management and subnet automation.
- **[LangCompatBridge](./lang-compat-bridge)**: Essential runtime for maintaining compatibility with older mod structures.

## 🤖 Gameplay Enhancements
- **[GregifyEmployees](./gregify-employees)**: Overhauls the NPC technician system with advanced AI and tiered skill progression.

## 🎓 Reference Examples
- **[UnlockAll](./unlock-all)**: A minimal Lua-based mod demonstrating how to hook into the game lifecycle and modify player stats.

---
*Note: All official mods are open-source and can be found in the [gregFramework GitHub Organization](https://github.com/mleem97).*
